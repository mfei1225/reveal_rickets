Rickets vs Osteomalacia

  • Rickets is deficient bone mineralization of growth plate (pediatric)
  • Osteomalacia is deficient bone mineralization of bone matrix (adults)

Etiology

  • Deficiency of vitamin D (diet, sun exposure, malabsorption)
  • Nutritional deficiency of calcium or phosphorus
  • Genetic causess
  • Drug induced causes
  • Secondary to liver diease

Adverse Events

  • Death
    • Untreated: up to 30%
    • Treated with anticoagulation: 2%-11%
  • Recurrent PE and DVT while on anticoagulant therapy
    • 6% in first 3 months
  • Chronic thromboembolic pulmonary hypertension
    • 1%-4% of acute PE survivors after 3 months to 2 years

Steps to diagnosis

  • Assess quality of exam
  • Determine if there is a PE

Pulmonary Motion Artifact

  • Blurry vessels on lung windows in lingula and/or lower lobes
  • This can be due to cardiac motion (near the heart) or respiratory motion (near the diaphragm)

Flow Related Artifact

  • Ill-defined margins (can be "wispy" looking) due to incomplete mixing of contrast and blood.
  • If ROI >78 HU on standard PE CT, this is typically artifact and not acute PE

Benefits of reveal.js+pacsbin

  • Presentations are automatically websites so they naturally work well for both giving a talk but also providing an easily accessible reference material.
  • Vertical slide feature allows creating "detail"/"supplementary" slides which can be great for diving deeper into topics/giving extra cases either based on real-time audience feedback or as material for the audience to do offline.
  • By using reveal.js, it is much easier to integrate interactive tools like pacsbin. This has the opportunity to make lectures significantly more interactive and more powerful than watching a youtube video.
  • Reveal.js provides the option to preload all of the websites being used in a presentation so this way all of the pacsbin cases wouldn't need to load once you clicked on a link.
  • Although the fact that this is built with code may be a drawback, moving away from powerpoint has interesting implications for collaboration that can be explained in detail in person.

Limitations of pacsbin

  • Pacsbin is awesome but it's not perfectly designed for embedding in lectures. We can do better!
  • A lightweight wrapper client can be built to provide more optimal presentation interaction.
  • This allows embedding different layouts while preserving screen real-estate
